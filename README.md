# FIFA-World-Cup-2026-Social-Media-Analytics

# Project OVERVIEW

The FIFA World Cup is one of the world's largest sporting events, generating millions of social media interactions across multiple platforms. Understanding fan engagement, public sentiment, platform performance, and player popularity can help marketing teams, broadcasters, sponsors, and event organizers make data-driven decisions.

This project presents an interactive Power BI dashboard that analyzes FIFA World Cup 2026 social media data, transforming raw datasets into meaningful business insights through KPIs, charts, filters, and interactive visualizations.

---

## Project Objective

The primary objective of this dashboard is to:

- Monitor social media engagement during the FIFA World Cup 2026.
- Analyze audience sentiment towards matches and teams.
- Identify the most active social media platforms.
- Discover trending players and emotions.
- Provide interactive insights for stakeholders through a single dashboard.

---

## Business Problem

Sports organizations receive enormous volumes of social media data every day. Without visualization, it becomes difficult to answer questions such as:

- Which platform generates the highest engagement?
- Which players are trending?
- What emotions dominate fan reactions?
- Which host cities receive the most online attention?
- How does engagement change across matches?
- What is the overall public sentiment?

This dashboard addresses these challenges by converting raw data into interactive business intelligence.

---

## Datasets Used

The project combines multiple datasets to build a complete analytical model.

### 1. merged_master.csv
Main fact table containing social media metrics.

### 2. social_media_posts.csv
Contains post-level information from multiple social media platforms.

### 3. match_context.csv
Contains FIFA match information including match dates and participating teams.

### 4. stakeholders.csv
Contains stakeholder-related information.

### 5. host_city_context.csv
Contains host city information used for geographical analysis.

---

## Data Model

Relationships were created between datasets using:

- Match ID
- Stakeholder ID
- Host City

This star-schema model enables efficient filtering and interactive reporting.
